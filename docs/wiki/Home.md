# Home

## Overview

Key Vault Explorer is a desktop app for browsing Azure Key Vault secrets with a local Azure CLI session.

The goal is to make day-to-day secret inspection easier for engineers who already work with `az login`.

## MVP Highlights

- Select an Azure subscription
- Discover Key Vault resources
- Browse secret names
- Inspect secret versions
- View secret values
- Copy secret values to the clipboard
- Search within the loaded secret list

## Audience

- Platform engineers
- Cloud engineers
- Developers working with Azure environments
- Teams that want a lightweight local Key Vault explorer without a custom auth flow
